  • Admins

    With Rust, you change advanced server settings from within the game. First, you must add yourself as an admin with your 17-digit SteamID64. Restarts take about 10-15 minutes.

    • Your Steam ID can be found in the URL of your Steam profile page. If you're using a custom URL, then paste that URL into Steam ID Finder.

      Find Steam ID

    • After becoming an admin, you can enter commands in-game by pressing F1 to access the console.

      View admin commands

  • Scheduler

    Schedule commands (restart or RCON) to run on your server. Servers left running too long are at risk to crash. This is particularly an issue with heavily modded servers. To keep your server healthy, it is wise to schedule regular restarts.
